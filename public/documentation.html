Movie API Application

This Movie API is built with REST architecture. It returns JSON responses, has self-explanatory URL's, and uses standard HTTP


Supported Requests

Request URL HTTP Method Request body data format Response body data format
Get list of all movies /movies GET None An array of JSON objects holding data about all the movies
Get list of movies by director /movies/directors/[director name] GET None An array of JSON objects holding data about all the movies by requested director
Get list of movies by year of release /movies/releaseYear/[year] GET None An array of JSON objects holding data about all the movies by requested year of release
Get list of movies by genre /movies/genres/[genre] GET None An array of JSON objects holding data about all the movies by requested genre
Get list of users /users GET None An array of JSON objects holding data about all the movies by requested genre
Get a specific user /users/userName GET None An array of JSON objects holding data about all the movies by requested genre
Get data about a single movie by title /movies/[title] GET None A JSON object holding data about the requested movie like this:
{ 'id': '######',
'title': 'The Batman',
'director': 'Matt Reeves',
'release-year': '2022',
'genre': 'Crime/Drama' }
Add a movie by ID users/userName/movies/moviedID POST A JSON object holding data about the movie to add, structured like:
{ 'title': 'The Batman',
'director': 'Matt Reeves',
'release-year': '2022',
'genre': 'Crime/Drama' } 		A JSON object holding data about the student that was added, including an ID:
{ 'id': '######',
'title': 'The Batman',
'director': 'Matt Reeves',
'release-year': '2022',
'genre': 'Crime/Drama' }
Remove a movie by ID /movies/id/[id] DELETE None A text message indicating whether the movie was successfully removed
Add a user /users POST A JSON object holding data about the user to add, structured like:
{ 'username': '[username]',
'password': 'PASSWORD',
'Birth-date': '01/01/1980',
'favorited-movies':
{ 'movie': '[movie id]'} } 		A JSON object holding data about the user that was added, including an ID:
{ 'id': '######',
'username': '[username]',
'password': '{password}',
'Birth-date': '01/01/1980',
'favorited-movies':
{ 'movie': '[movie id]'} }
Remove a movie from a user's favorites list /users/[name]/[movieID] DELETE None A text message indicating whether the movie was successfully removed from the user's favorited movies
